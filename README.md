# servicenow-itsm-incident-management
Hands-on ServiceNow ITSM practice project demonstrating incident and request management workflows.

## **Project Overview**

This project simulates and IT Service Management workflow inside the ServiceNow Service Operations Workspace.
It includes creating, updating, and resolving incidents as well as fulfilling a service request.

## **Key Activities Completed**

- Managed multiple incidents with different priority levels
- Added work notes, communication updates, and resolution documentation
- Resolved SLA warning and worked with breached SLAs
- Investigated and resolved a critical incident 
- Completed a service request through the request fulfillment workflow
- Followed standard ITIL practices for incident and request handling
